> "Programming becomes powerful when your code starts making decisions instead of just executing instructions."

# 📌 Overview

Today marks the beginning of **Decision Making** in Dart.

Until now, our programs followed a simple sequence:

Input → Process → Output

From today onwards, the program can **think before acting** by checking a condition.

The `if` statement allows the program to execute code **only when a condition is true**.

This is one of the most fundamental concepts in programming and is widely used in every software application.

# ☕ Java vs Dart

### Java

```java
if(age >= 18){
    System.out.println("Eligible");
}
```

### Dart

```dart
if(age >= 18){
    print("Eligible");
}
```

The syntax is almost identical, making it easy for Java developers to transition to Dart.

# 💡 Key Concept

The `if` statement executes a block of code **only when the given condition evaluates to true**.

If the condition is false, the program simply skips that block and continues executing the remaining code.

---

# 🚀 Developer Tips

✔ Always write meaningful conditions.

✔ Prefer curly braces `{}` even for a single statement.

✔ Keep conditions simple and readable.

✔ Write code that explains itself.

✔ Think about all possible scenarios while writing conditions.
