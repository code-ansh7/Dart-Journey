# 🚀 Day 10 - switch Statement & Ternary Operator

> "Choose the right tool for the right decision."

---

# 📖 Overview

Day 10 focused on two important decision-making concepts in Dart:

- switch Statement
- Ternary Operator

Since I already had experience with `switch` from Java, this day focused on applying it in real-world scenarios instead of repeating basic exercises. The Ternary Operator was introduced as a concise alternative to simple `if-else` statements.

---

# 🎯 Objectives

- Learn when to use `switch`
- Understand the limitations of `switch`
- Learn the Ternary Operator
- Write cleaner and shorter conditional expressions
- Build interview-oriented programs

---

# 📚 Topics Covered

- switch Statement
- break Statement
- default Case
- Ternary Operator (`condition ? true : false`)
- Logical OR (`||`) with Ternary
