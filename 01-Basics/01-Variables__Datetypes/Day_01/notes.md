# 📝 Dart Day 1 Notes

## Dart

- Dart is a programming language developed by Google.
- Flutter uses Dart to build cross-platform applications.

---

## Dart File

Extension:

```text
.dart
```

Example:

```text
main.dart
```

---

## main()

- Entry point of every Dart program.
- Execution always starts from `main()`.

Syntax:

```dart
void main() {

}
```

---

## void

- Return type of the function.
- Means the function does not return any value.

---

## print()

Used to display output on the console.

Example:

```dart
print("Hello Dart");
```

---

## Line Termination

Every statement ends with a semicolon (`;`).

Example:

```dart
print("Hello");
```

---

## Java vs Dart

Java

```java
System.out.println("Hello");
```

Dart

```dart
print("Hello");
```

---

## Flutter Connection

Flutter application execution starts from `main()`.
