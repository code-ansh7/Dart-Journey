## 🧠 Key Learnings

- Every Dart program starts from `main()`.
- `print()` displays output on the terminal.
- `void` means the function returns nothing.
- `;` marks the end of a statement.
- Dart syntax is cleaner than Java.

---

## 📱 Flutter Connection

Flutter applications also start execution from the `main()` function.

Example:

```dart
void main() {
  runApp(MyApp());
}
```

Without understanding `main()`, Flutter's entry point becomes confusing.
