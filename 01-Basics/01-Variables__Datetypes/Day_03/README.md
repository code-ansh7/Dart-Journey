## 🧠 What I Learned

- String Interpolation is cleaner than string concatenation.
- `var` allows the compiler to infer the data type.
- `var` does NOT mean dynamic.
- Once the compiler infers a type, it cannot be changed.
- Type Safety prevents invalid assignments.
- Console output should be user-friendly and properly formatted.

---

## ☕ Java vs Dart

### Java

```java
System.out.println("Age : " + age);
```

### Dart

```dart
print("Age : $age");
```

---

## 📱 Flutter Connection

Flutter developers use:

- String Interpolation
- var
- Type Inference

almost everywhere while displaying UI data.

Example:

```dart
Text("Welcome $userName")
```
