## 🧠 What I Learned

- Variables store data in memory.
- Every variable has a data type.
- Different data types store different kinds of values.
- `print()` displays variable values.
- Non-string values must be converted using `toString()` when concatenating with a string.
- Meaningful variable names improve code readability.

---

## ☕ Java vs Dart

Java

```java
String name = "Ansh";
System.out.println("Age : " + age);
```

Dart

```dart
String name = "Ansh";
print("Age : " + age.toString());
```
