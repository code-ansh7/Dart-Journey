## Day 2

### Variable

A variable stores data in memory.

Syntax

```dart
DataType variableName = value;
```

---

### Data Types

- String → Text
- int → Whole Numbers
- double → Decimal Numbers
- bool → true / false

---

### Printing Variables

```dart
print(variableName);
```

---

### Type Conversion

```dart
value.toString()
```

Used while concatenating strings with non-string values.

---

### Naming Rules

- Use meaningful names.
- Use camelCase.
- No spaces.
- Cannot start with a number.
