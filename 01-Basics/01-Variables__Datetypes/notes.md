### String Interpolation

```dart
print("Age : $age");
```

### Expression

```dart

```

### var

Compiler automatically identifies the type.

```dart
var name = "Ansh";
```

### Type Inference

```
var age = 19

↓

Compiler

↓

int age = 19
```

### Type Safety

Once inferred

↓

Cannot change data type.
