# ☕ Java vs Dart

Most operators work exactly like Java.

Example

```java
int sum = a + b;
boolean result = age >= 18;
```

```dart
var sum = a + b;
var result = age >= 18;
```

The main difference is Dart's `~/` operator for integer division.
