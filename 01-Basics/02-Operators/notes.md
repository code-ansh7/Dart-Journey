# 📒 Operators

---

# 📖 Introduction

Operators are one of the most important concepts in programming.

Without operators we cannot:

- Calculate Bills
- Calculate Marks
- Find Profit/Loss
- Calculate Interest
- Compare Values
- Make Decisions

Almost every real-world software uses operators.

---

# ❓ What is an Operator?

## Definition

An operator is a special symbol that performs an operation on values or variables.

Example

```dart
10 + 20
```

Here,

```
+
```

is an operator.

---

# 🌍 Real Life Example

Calculator

```
15 + 5

↓

20
```

Shopping App

```
Price × Quantity

↓

Total Bill
```

Banking App

```
Balance - Withdraw

↓

Remaining Balance
```

# 📚 Day 4 Summary

## Assignment Operator

```
=
```

Purpose

Assigns values to variables.

Example

```dart
var age = 19;
```

---

## Arithmetic Operators

| Operator | Purpose |
|----------|----------|
| + | Addition |
| - | Subtraction |
| * | Multiplication |
| / | Division |
| ~/ | Integer Division |
| % | Modulus |

---

# Difference Between `/` and `~/`

```
10 / 3

↓

3.333333
```

```
10 ~/ 3

↓

3
```

---

# Modulus

```
10 % 3

↓

1
```

Returns the remainder.

# 💡 Best Practices

✅ Use meaningful variable names.

✅ Store calculation results inside variables.

✅ Format console output properly.

✅ Avoid printing direct calculations repeatedly.

---

# ⚠ Common Mistakes

❌ Confusing `/` with `~/`

❌ Poor variable naming

❌ Spelling mistakes

❌ Printing calculations directly instead of storing them

---

# ☕ Java vs Dart

Almost all arithmetic operators work exactly like Java.

The only new operator for Java learners is

```
~/
```

which performs Integer Division.
