# Arithmetic Operators

| Operator | Meaning | Example |
|----------|---------|----------|
| + | Addition | 10 + 5 |
| - | Subtraction | 10 - 5 |
| * | Multiplication | 10 * 5 |
| / | Division (Returns double) | 10 / 5 |
| ~/ | Integer Division | 10 ~/ 5 |
| % | Modulus (Remainder) | 10 % 3 |

---

# Comparison Operators

| Operator | Meaning |
|----------|---------|
| > | Greater Than |
| < | Less Than |
| >= | Greater Than or Equal |
| <= | Less Than or Equal |

Return Type

```dart
bool
```

# Compound Assignment

| Operator | Equivalent |
|----------|------------|
| += | a = a + b |
| -= | a = a - b |
| *= | a = a * b |
| /= | a = a / b |
| %= | a = a % b |

---

# Important Rules

✅ `/` always returns **double**

✅ `~/` always returns **int**

✅ Comparison Operators return **bool**

✅ Logical Operators work only with **boolean expressions**
