## 🧠 What I Learned

- Operators perform operations on variables and values.
- `/` always returns a `double`.
- `~/` removes the decimal part.
- `%` returns the remainder.
- Real-world calculations can be implemented using arithmetic operators.
- Proper console formatting improves readability.

---

## ☕ Java vs Dart

### Java

```java
int sum = a + b;
```

### Dart

```dart
var sum = a + b;
```
