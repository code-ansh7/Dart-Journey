# 🧠 What I Learned

- Multiple conditions can be combined using Logical Operators.
- `&&` requires all conditions to be true.
- `||` requires at least one condition to be true.
- `!` reverses a boolean value.
- Compound assignment operators update variables in a shorter and cleaner way.

---

# ☕ Java vs Dart

Logical and compound assignment operators behave exactly like Java.

One important point:

`/` always returns a `double` in Dart.
