# 🤔 What is Input?

Input means **the data given by the user to the program**.

Example:

```

Enter Your Name :

Ansh

```

Here,

```

Ansh

```

is the **Input**.

---

# 🤔 What is Output?

Output means **the information displayed by the program**.

Example

```

Welcome Ansh

```

This message is the Output.

# ☕ Java vs Dart

Java

```java
Scanner sc = new Scanner(System.in);

String name = sc.nextLine();
```

Dart

```dart
import 'dart:io';

String name = stdin.readLineSync()!;
```
