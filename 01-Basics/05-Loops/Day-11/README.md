# 🚀 Day 11 - for Loop (Real World Programs)

> "Loops eliminate repetition and make programs smarter."

---

# 📖 Overview

Day 11 introduced the `for` loop in Dart through practical, real-world examples instead of traditional beginner exercises.

Instead of printing numbers or patterns, this day focused on solving real application scenarios such as login attempts, OTP verification, password validation, and ATM systems.

This approach strengthened both programming logic and interview preparation.

---

# 🎯 Objectives

- Understand the purpose of loops
- Learn the syntax of the `for` loop
- Build real-world programs using loops
- Combine loops with conditional statements
- Improve logical thinking

---

# 📚 Topics Covered

- for Loop
- Loop Initialization
- Loop Condition
- Loop Update
- Combining `for` with `if-else`
- Combining `for` with `switch`
- Using `return` inside loops
