## 🎯 Objective

Learn how to pass data into functions using parameters and understand how functions become reusable by accepting different values.

- Why Parameters are Needed

## 💡 What I Learned

- Functions become reusable using parameters.
- Parameters allow one function to work with different data.
- Multiple parameters can be passed together.
- Functions help organize application logic.
- Controller functions make code cleaner and easier to understand.

## 📌 Key Takeaways

- Avoid writing duplicate code.
- Keep one responsibility for each function.
- Pass data through parameters instead of hardcoding values.
- Write meaningful function names.
- Think about application flow before writing code.
