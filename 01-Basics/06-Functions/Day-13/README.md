# 🧠 What I Learned

- A Function performs one specific task.
- Void Functions execute code without returning a value.
- Controller Functions organize application flow.
- Every Function should have only one responsibility.
- Large applications are built using multiple small Functions.
- Keeping `main()` clean improves readability and maintenance.
