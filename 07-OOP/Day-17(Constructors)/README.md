## 💡 What I Learned

- Constructors initialize objects.
- `this.field` automatically assigns constructor values to class fields.
- Named Parameters improve code readability.
- `required` makes important parameters mandatory.
- Optional parameters allow flexible object creation.
- Nullable variables (`?`) can store `null`.
- `??` provides a fallback value when a variable is `null`.
- Default Values remove the need to pass every parameter.
- Initializer Lists initialize or validate `final` fields before the constructor body executes.

---

## 📱 Flutter Connection

These constructor patterns are heavily used in Flutter widgets like:

- Text()
- Container()
- Scaffold()
- Padding()
- Card()
- ElevatedButton()
- ListTile()

Understanding constructors makes Flutter widgets much easier to understand.
