# 🧠 Concepts Learned

### ✔ Returning Numbers

Functions can return numeric values like:

- int
- double

instead of only printing results.

---

### ✔ Returning Boolean

Functions can return

- true
- false

which is useful for validations.

---

### ✔ Returning Strings

Functions can generate complete reports, invoices, or messages and return them instead of printing directly.

---

### ✔ Function Chaining

The output of one function can become the input of another function.

Example:

calculateTotal()

↓

calculateDiscount()

↓

calculateFinalAmount()

↓

showDetails()

---

### ✔ Reusability

Instead of writing calculations multiple times, reusable functions were created.

---

### ✔ Clean Function Design

Each function had only one responsibility.

Examples:

- calculateTotal()
- calculateDiscount()
- calculateFinalAmount()
- showDetails()
