# 🧠 Concepts Learned

## ✔ Code Reusability

Functions help avoid duplicate code.

Instead of writing the same logic multiple times, one reusable function can be called whenever needed.

---

## ✔ Separation of Responsibility

Each function should perform only one task.

Example:

calculateTotal()

Only calculates total.

printBill()

Only displays bill.

---

## ✔ Better Readability

Small functions make programs easier to understand.

---

## ✔ Better Maintenance

If business logic changes, only one function needs to be updated.

---

## ✔ Better Testing

Functions can be tested independently.

---

## ✔ Professional Coding Style

Large applications are built by combining multiple small functions.
