# 🚀 Dart Day 5 - Comparison & Equality Operators

## 🎯 Today's Goal

Understand how Dart compares values using Relational and Equality Operators. Learn how comparison operations return boolean values (`true` or `false`) and how these operators are used in real-world applications.

---

# 📚 Topics Covered

## Relational Operators

- Greater Than (`>`)
- Less Than (`<`)
- Greater Than or Equal To (`>=`)
- Less Than or Equal To (`<=`)

---

## Equality Operators

- Equal To (`==`)
- Not Equal To (`!=`)

# 🧠 What I Learned

- Comparison operators compare two values.
- Equality operators check whether two values are equal.
- Every comparison returns a boolean value.
- Comparison results can be stored inside variables.
- Boolean values are the foundation of decision making.

---

# ☕ Java vs Dart

Java

```java
age >= 18
```

Dart

```dart
age >= 18
```

Comparison operators work exactly the same in both languages.

---

# 📱 Flutter Connection

Comparison operators are used in almost every Flutter application.

Examples:

- Login Validation
- OTP Verification
- Age Verification
- Student Eligibility
- Shopping Offers
- Free Delivery
- Permission Checks

Without comparison operators, Flutter apps cannot make decisions.
