# 📘 Dart Day 4 Notes

# Operators

## Definition

Operators are special symbols used to perform operations on values or variables.

---

# Why Operators?

Without operators, calculations like:

- Total Price
- Average Marks
- Remaining Balance
- Interest Calculation

cannot be performed.

---

# Types Covered Today

## Assignment Operator (=)

Assigns a value to a variable.

Example

```dart
var age = 19;
```

---

## Addition (+)

Adds two values.

```dart
var sum = a + b;
```

---

## Subtraction (-)

Subtracts one value from another.

```dart
var result = a - b;
```

---

## Multiplication (*)

Multiplies two values.

```dart
var total = price * quantity;
```

---

## Division (/)

Returns a decimal value.

```dart
10 / 3

Output

3.333333...
```

---

## Integer Division (~/)

Returns only the integer part.

```dart
10 ~/ 3

Output

3
```

---

## Modulus (%)

Returns the remainder.

```dart
10 % 3

Output

1
```

# Interview Questions

Q. Difference between `/` and `~/`?

Ans.

`/` returns a double.

`~/` returns an integer.
